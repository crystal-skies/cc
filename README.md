# Hi
Mostly a very simple way to see how trades and the basis of investing could actually work. But as this project is essentially a very low-accuracy project, it is just to play with some variables and try using ML to make good predictions.

In this project I achieved numbers like 74% accuracy with a combination of an RSI alert that passes through an ML (random forest) that is pretrained every week, and finally the risk chip, which would let me say how much should I risk (0 to n) and in which way.

Anyways, this was a bit of a disordered project. The numbers I got could indicate overfitting. I realized by searching that most of the best numbers for a prediction or an adaptable strategy were around 51% to 60%, so that's where I realized that my 74% was actually overfitting. Just taking some 15-minute candles and a short number of features and just a single ML will not work, that's why I decided to do something different in the same direction.


# Hola
Este proyecto es , en su mayoría, una forma muy simple de ver cómo podrían funcionar realmente las operaciones y las bases de la inversión. Pero como este proyecto es esencialmente un proyecto de muy baja precisión, es solo para jugar con algunas variables e intentar usar ML para hacer buenas predicciones.

En este proyecto logré obtener números como 74% de precisión con una combinación de una alerta de RSI que pasa por un modelo de ML (random forest) que se preentrena cada semana y, finalmente, el risk chip, que me permitiría decir cuánto debería arriesgar (0 a n) y de qué manera.

De todos modos, este fue un proyecto un poco desordenado. Los números que obtuve podrían indicar sobreajuste. Me di cuenta al investigar que la mayoría de los mejores números para una predicción o una estrategia adaptable estaban alrededor de 51% a 60%, así que ahí fue cuando entendí que mi 74% era en realidad sobreajuste. Solo tomar velas de 15 minutos, una cantidad pequeña de variables y un solo modelo de ML no va a funcionar, por eso decidí hacer algo diferente en la misma dirección.
